# Jarvis-OS : Personal AI Architect & Chaos Engine 🧠🌀

A collection of Python tools and scripts I built as a Computer Science student at EPFL to automate my daily workflow, challenge my learning habits, and break out of algorithmic echo chambers.

## Features

### 1. The Chaos Engine (`chaos_engine.py`)
An anti-echo-chamber script designed to force intellectual serendipity. 
It randomly pulls advanced research papers from ArXiv (Neuroscience, AI, Crypto) and pairs them with completely unrelated Wikipedia articles, prompting the user to find cross-disciplinary connections.

### 2. University API Scraper (`morning_brief.py`)
An automated morning briefing system. It securely authenticates with the university's Ed Discussion API, parses the newest posts and announcements from my specific courses (Algorithms, Machine Learning, Neuro-engineering), and aggregates them into a single daily digest.

### 3. Socratic AI Tutor System (WIP)
A strict, prompt-engineered system constraint. Instead of giving direct code solutions, the AI is constrained to act as an elite EPFL professor, asking guiding questions and utilizing the Socratic method to teach C Programming and Machine Learning.

## Tech Stack
- **Python 3** (Requests, ElementTree)
- **APIs:** ArXiv API, Wikipedia REST API, Ed Discussion API.
